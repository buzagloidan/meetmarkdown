# MeetMarkdown

Free online markdown tools — no sign-up, no upload, no tracking.

## Tools

- **Live Editor** — real-time markdown preview with Mermaid diagrams
- **Formatter** — prettify markdown with Prettier
- **Markdown to HTML** — convert to clean semantic HTML
- **Table Formatter** — align GFM tables
- **HTML to Markdown** — convert HTML to markdown via Turndown
- **Markdown to PDF** — export via browser print dialog
- **Word Count & Stats** — reading time, headings, and more
- **Markdown Diff** — compare two documents line by line
- **URL to Markdown** — fetch any public URL as markdown
- **Mermaid to Image** — export Mermaid diagrams as PNG/JPG

## Stack

Next.js 16 · TypeScript · Tailwind CSS · shadcn/ui · Mermaid · Prettier

## Development

```bash
npm install
npm run dev
```
